# Project Riko

![Project Riko](https://img.shields.io/badge/Project-Riko-purple?style=for-the-badge)

**Live Demo:** https://spadesreduxed.github.io/riko_project/

Your AI Anime Companion.

Riko is an anime-focused LLM project by Just Rayen. She listens, remembers your conversations, and responds in a snarky anime girl voice — powered by GPT-4.1-mini, GPT-SoVITS voice synthesis, and Faster-Whisper ASR.

## Features

- 💬 LLM Dialogue
- 🧠 Memory
- 🔊 Voice Synthesis
- 🎧 ASR
- ⚙️ YAML Config

## Tech Stack

- OpenAI GPT
- GPT-SoVITS
- Faster-Whisper

## Quick Start

```bash
pip install uv
uv pip install -r requirements.txt
python main_chat.py
```

> Requires Python 3.10+ · Windows 10+ or Linux Ubuntu · Microphone & speakers recommended
